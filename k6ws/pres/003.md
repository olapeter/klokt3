# Oppsett

1. Installer K6 (https://grafana.com/docs/k6/latest/set-up/install-k6/)
    - Win: `winget install k6 --source winget`
    - Mac: `brew install k6`
    - Linux: det vet du best selv
1. Lag ei ny mappe som heiter `k6ws`
1. Inne i mappa, kjør `k6 new`
